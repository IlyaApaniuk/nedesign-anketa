# NE DESIGN — анкета клиента

Пошаговая веб-анкета для студии дизайна интерьера NE DESIGN (Новикова Елена).
Сделана по мотивам PDF-брифа `Modern_Black_&_White_Fashion_Brand_Portfolio_Presentation.pdf`.

## Что умеет

- 32 шага: общие данные → 34 открытых вопроса → материалы и комплектация
  (с фото-подсказками из PDF) → референсы стилей по комнатам → палитра цветов
- Черновик автоматически сохраняется в localStorage браузера — можно закрыть
  вкладку и продолжить позже
- Адаптив под телефон
- В конце кнопка «Скачать ответы (PDF)» генерирует настоящий PDF прямо
  в браузере (jsPDF + вшитый Montserrat с кириллицей): в файл попадают
  только заполненные ответы, выбранные фото-референсы и цвета. Клиент
  отправляет его в Telegram `@nov_alyonka`. Бэкенда нет.

## Файлы

- `index.html`, `styles.css`, `app.js` — приложение
- `data.js` — вся структура анкеты (вопросы, опции, цвета). Править тексты — здесь
- `fonts.css` — Montserrat (latin + cyrillic) как data-URI, работает офлайн
- `jspdf.umd.min.js`, `pdffont.js` — генерация PDF с ответами
- `img/` — фото, извлечённые из PDF (90 референсов стилей + 101 фото опций)
- `img_small/` — уменьшенные копии для однофайловой сборки
- `build_single.py` — собирает всё в один файл `dist/anketa.html`
  (картинки инлайнятся): `python3 build_single.py`
- `dist/anketa.html` — самодостаточный файл: работает без сервера,
  можно просто отправить клиенту или залить на любой хостинг

## Запуск локально

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Как выложить бесплатно

Любой вариант даёт постоянную ссылку:

1. **Netlify Drop** — drop.netlify.com, перетащить папку проекта (без `dist/`
   и PDF), получить ссылку. Самый быстрый способ.
2. **GitHub Pages** — создать репозиторий, залить файлы, включить Pages
   в настройках (branch `main`, folder `/`).
3. **Vercel** — `npx vercel` в папке проекта.

Либо вообще без хостинга: отправлять клиенту файл `dist/anketa.html` —
он открывается локально двойным кликом.
